# randomKey

## 项目描述

随机密码生成器

## 技术栈

HTML
CSS
JavaScript(原生Ajax)
Python
Flask(Web后端服务)
Docker(容器端口映射)
Supervisor(进程管理)
Nginx(反向代理)

## Build Setup

```bash

# 安装依赖
pip3 install -r requirements.txt

# 启动Web App
python begin.py

```
