# randomKey

> 随机密码生成器

## Tech Stack

- HTML/CSS/JavaScript
- Python/Flask
- Docker
- Supervisor
- Nginx

## Build Setup

```bash
# 安装依赖
pip3 install -r requirements.txt

# 启动Web App
python begin.py
```
